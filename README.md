# Bristol Clean Air Zone (CAZ) Evaluation Dashboard

This repository contains the source code and data for the MSc Dissertation: 
**"Evaluating the Bristol Clean Air Zone: An Interactive Spatial-Temporal Analysis"**.

### 🔗 Live Dashboard
**Access the app here:** https://bristol-caz.streamlit.app

### 📂 Repository Structure
* **`Home.py`**: The main executive dashboard.
* **`pages/`**: Detailed analysis scripts for Causal Impact and Spatial Mapping.
* **`data_processing_scripts/`**: Python scripts used to clean the raw pollution data.
* **`r_analysis/`**: The R code used for the Bayesian Structural Time-Series models.
